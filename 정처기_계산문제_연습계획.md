## 1. CPU 스케줄링 계산

**출제 유형:** RR, SJF, SRT 평균 대기시간 / 반환시간 계산

**핵심 공식:**
- 대기시간 = 완료시간 - 도착시간 - 실행시간
- 반환시간 = 완료시간 - 도착시간
- 평균 대기시간 = 각 프로세스 대기시간 합 / 프로세스 수

**연습 목표:**
- [ ] RR (타임퀀텀 주어짐): 간트 차트 그리고 대기시간 계산
- [ ] SJF (비선점): 실행시간 짧은 순서로 계산
- [ ] SRT (선점형 SJF): 남은 시간 기준 계산
- [ ] FCFS: 도착 순서대로 계산

## 2. 서브넷 마스크 계산

**출제 유형:** IP 주소 + 서브넷 마스크 → 네트워크 주소 / 호스트 수

**핵심 공식:**
- 네트워크 주소 = IP AND 서브넷 마스크 (비트 연산)
- 호스트 수 = 2^(32-n) - 2 (네트워크 주소, 브로드캐스트 주소 제외)
- /24 = 255.255.255.0 → 호스트 수: 254
- /26 = 255.255.255.192 → 호스트 수: 62

**연습 목표:**
- [ ] CIDR 표기 (/n) → 서브넷 마스크 변환
- [ ] IP AND 서브넷 마스크 → 네트워크 주소
- [ ] 가능한 호스트 수 계산

## 3. 소프트웨어공학 계산

### 3-1. 순환 복잡도 (Cyclomatic Complexity)
**핵심 공식:** V(G) = E - N + 2P
- E: 간선 수, N: 노드 수, P: 연결된 요소 수 (보통 1)
- 또는 V(G) = 결정 포인트 수 + 1

**연습 목표:**
- [ ] 제어 흐름 그래프 주어지면 E, N 세어서 계산

### 3-2. LOC 기법
**핵심 공식:**
- 예측 LOC = (낙관치 + 4×기대치 + 비관치) / 6
- 개발 기간 = LOC / (개발자 수 × 1인당 월 LOC)
- 노력(MM) = LOC / 생산성

**연습 목표:**
- [ ] 낙관/기대/비관치 주어지면 예측 LOC 계산
- [ ] LOC → 개발 기간(MM) 계산

### 3-3. FP (Function Point)
**핵심 공식:**
- FP = 기능 점수 합계 × (0.65 + 0.01 × 총 영향도)
- 5가지 기능: 외부 입력/출력/조회/내부 논리파일/외부 인터페이스 파일

**연습 목표:**
- [ ] 각 기능에 가중치 곱해서 FP 계산

### 3-4. Man Month (MM)
**핵심 공식:**
- MM = 개발 기간 × 개발 인원
- 개발 기간 = MM / 인원
- 생산성 = LOC / MM

**연습 목표:**
- [ ] LOC, 인원, 기간 중 2가지 주어지면 나머지 계산

## 4. SQL 계산

**출제 유형:** 집계 함수 결과값 / 조건 필터링 후 COUNT

## 6. 페이지 교체 알고리즘 (OS)

**출제 유형:** 페이지 참조열 주어지면 페이지 부재(Page Fault) 횟수 계산

**핵심 알고리즘:**
- FIFO: 먼저 들어온 것 교체
- LRU (Least Recently Used): 가장 오래전에 사용된 것 교체
- LFU (Least Frequently Used): 사용 횟수 가장 적은 것 교체
- OPT (Optimal): 앞으로 가장 오래 사용 안 될 것 교체 (이론값)

**연습 목목표:**
- [ ] 참조열 주어지면 FIFO/LRU/LFU 각각 page fault 횟수 계산

## 7. 비트 연산 (주로 코드 추적 안에 포함)

**핵심 연산:**
- `&` (AND): 둘 다 1이면 1
- `|` (OR): 하나라도 1이면 1
- `^` (XOR): 다르면 1
- `~` (NOT): 반전
- `<<` (좌시프트): ×2
- `>>` (우시프트): ÷2

**연습 목표:**
- [ ] C 코드에서 비트 연산 결과 계산
- [ ] 진수 변환 후 비트 AND/OR 결과 계산

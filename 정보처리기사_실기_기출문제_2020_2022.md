### 자주 출제된 핵심 개념

#### SQL (★★★)
- SELECT / WHERE / GROUP BY / HAVING / ORDER BY
- JOIN (INNER JOIN, CROSS JOIN)
- INSERT / UPDATE / DELETE
- CREATE TABLE / ALTER TABLE / CREATE INDEX
- 집계 함수: COUNT, MIN, MAX, AVG
- 서브쿼리, IN / ALL / ANY

#### 프로그래밍 (★★★)
- Java: 상속(extends), 오버라이딩, static, super, 재귀
- C언어: 포인터, 구조체, 재귀, 비트 연산
- Python: 리스트 메서드, 람다, 집합 연산, 비트 연산

#### 데이터베이스 (★★★)
- 정규화: 1NF, 2NF, 3NF, BCNF (부분/이행 함수 종속)
- 트랜잭션 ACID (원자성, 일관성, 독립성, 지속성)
- 회복 기법: REDO, UNDO, 즉각갱신, 지연갱신
- 관계 대수: SELECT(σ), PROJECT(π), JOIN(⋈), DIVISION(÷)
- 이상 현상: 삽입 이상, 삭제 이상, 갱신 이상
- Cardinality / Degree / 슈퍼키 / 후보키

#### 소프트웨어공학 (★★★)
- 결합도(낮음 우선): 자료 < 스탬프 < 제어 < 외부 < 공통 < 내용
- 응집도(높음 우선): 기능 > 순차 > 교환 > 절차 > 시간 > 논리 > 우연
- 테스트: 블랙박스(경계값, 동등분할, 원인-효과), 화이트박스(문장/결정/조건 커버리지)
- 디자인 패턴: 생성(Factory Method, Singleton) / 구조(Bridge) / 행위(Observer, Command)
- V-모델: 단위→통합→시스템→인수 테스트
- Fan-in / Fan-out

#### 보안 (★★)
- 암호화: AES, DES, MD5, SHA-1, TKIP, IDEA, SKIPJACK
- 공격 기법: LAND Attack, 스니핑, ARP 스푸핑, SQL Injection, 세션 하이재킹, Watering Hole
- 보안 3요소: 기밀성, 무결성, 가용성
- 접근제어: DAC, MAC, RBAC
- 인증: IPSec, SSO, ISMS

#### 네트워크 (★★)
- OSI 7계층, TCP/IP 4계층
- 라우팅 프로토콜: OSPF, RIP, BGP, EGP, IGP
- IPv4 / IPv6, 서브네팅, NAT, VPN
- 패킷 교환: 가상 회선 방식, 데이터그램 방식
- 프로토콜: ICMP, SOAP, WSDL, RARP, ARP, IPSec
